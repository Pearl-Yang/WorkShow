# 个人作品集 - 带水印保护

## 项目简介

一个现代化的个人作品集网站，展示个人作品的同时，通过技术手段在用户尝试保存或复制时自动添加水印。

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript（无需后端）
- Canvas API 实现图片水印
- CSS/JS 防护措施

## 核心功能

### 1. 水印保护机制

- **Canvas 渲染**：所有图片通过 Canvas 渲染，原始图片地址不可直接访问
- **动态水印**：在 Canvas 上绘制图片并叠加半透明水印
- **禁用右键**：阻止右键菜单和图片保存
- **防复制**：禁止选中文本和拖拽图片
- **复制保护**：用户复制图片时会粘贴带水印的版本

### 2. 作品展示

- 响应式网格布局
- 分类筛选功能
- 点击放大查看
- 平滑动画效果

### 3. 个人信息

- 个人简介
- 联系方式
- 社交链接

## 使用方法

直接在浏览器中打开 `index.html` 文件即可。

## 注意事项

- 水印仅为防护手段，无法完全防止专业截图
- 建议配合版权声明使用
